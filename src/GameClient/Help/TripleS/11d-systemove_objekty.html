Množina

Konstrukce promìnné typu množina viz Definice množiny.

Operace s množinami

Sjednocení, prùnik, rozdíl

TripleS obsahuje následující operátory pro sjednocení, prùnik a rozdíl množin. Množiny, na kterých má být daná operace provedena, musí být stejného typu.

+sjednocení
*prùnik
-rozdíl

Dotaz na prvek

Pro zjištìní, zda je nìjaká hodnota prvkem množiny, je v TripleS operátor in. Na jeho pravé stranì musí být promìnná typu množina, na levé stranì hledaná hodnota stejného typu, jako jsou prvky množiny. Výsledek je typu bool, hodnota je pak true pokud množina daný prvek obsahuje nebo false pokud ne.

Pøíklad použití:

char Znaky{};
...
if ( 'A' in Znaky ) { /* písmeno A je v mnozine znaky */ }

Enumerace

Pomocí for cyklu pro množiny lze enumerovat všechny prvky množiny. Viz for cyklus pro množinu.

Metody

int GetItemCount ()

Vrací poèet položek v množinì.

bool IsEmpty ()

Vrací true, je-li množina prázdná, jinak vrací false.
